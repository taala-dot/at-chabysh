# 🏇 Кыргызские Конные Гонки

Мультиплеерная игра на скорость печати с киргизским национальным стилем

## 🎮 Возможности

✅ **Мультиплеер** - до 4 игроков в одной комнате
✅ **Боты** - автоматически заполняют пустые места
✅ **Комнаты с кодом** - пригласите друзей
✅ **Обратный отсчет** - 3-2-1-GO перед стартом
✅ **Реальное время** - смотрите прогресс всех игроков
✅ **Статистика** - WPM, точность, время
✅ **Таблица лидеров** - с медалями и позициями
✅ **Киргизский дизайн** - цвета флага, орнаменты, национальный стиль

## 📦 Установка

### 1. Установите Node.js
Скачайте с [nodejs.org](https://nodejs.org/) (рекомендуется LTS версия)

### 2. Установите зависимости
```bash
cd horse-racer
npm install
```

### 3. Запустите сервер
```bash
npm start
```

Или:
```bash
node server.js
```

### 4. Откройте в браузере
```
http://localhost:3000
```

## 🎯 Как играть

1. **Введите свое имя**
2. **Создайте комнату** или **введите код** для присоединения
3. **Поделитесь кодом** с друзьями (остальные места заполнят боты)
4. **Нажмите "Начать гонку"** (только хост может стартовать)
5. **Печатайте текст** и обгоняйте соперников!
6. **Смотрите результаты** после финиша

## 🤖 Боты

Боты автоматически добавляются если игроков меньше 4:
- **Буран 🤖** - скорость ~65 WPM
- **Молния 🤖** - скорость ~55 WPM
- **Звезда 🤖** - скорость ~45 WPM
- **Гром 🤖** - скорость ~50 WPM

## 🎨 Киргизский стиль

- Цвета флага Кыргызстана (красный + желтый)
- Национальные орнаменты
- Конь 🏇 развернут в правильную сторону
- Киргизские тексты для набора

## 🛠️ Технологии

- **Node.js** - серверная часть
- **Express** - веб-сервер
- **Socket.io** - WebSocket связь в реальном времени
- **Vanilla JS** - клиентская часть (без фреймворков)

## 🌐 Деплой

Для деплоя на сервер (Heroku, Railway, DigitalOcean):

1. Установите переменную окружения `PORT`
2. Измените в `index.html`:
```javascript
const socket = io('http://localhost:3000');
```
на:
```javascript
const socket = io(); // автоопределение адреса
```

## 🐛 Решение проблем

**Сервер не запускается:**
- Проверьте что установлен Node.js: `node --version`
- Установите зависимости: `npm install`

**Не подключается к серверу:**
- Проверьте что сервер запущен
- Убедитесь что открыли `http://localhost:3000`
- Проверьте консоль браузера на ошибки

**Боты не появляются:**
- Боты добавляются только после нажатия "Начать гонку"
- Боты заполняют места с 2-го по 4-е если там нет игроков
